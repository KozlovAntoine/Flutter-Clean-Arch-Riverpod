# Tuto Riverpod Clean Architecture

Exemple d'application Flutter démontrant l'architecture Clean avec Riverpod, incluant la séparation
en couches (domain, data, presentation) et l'injection de dépendances.

## Prérequis

- Flutter SDK: `^3.38.5`
- Dart SDK: `^3.10.4`

## Commandes

```bash
# Générer les fichiers (mocks, freezed, riverpod, etc.)
dart run build_runner build --delete-conflicting-outputs
```
